Your personal AI gateway

Moltis connects LLMs to your messaging channels, tools, and devices. Written in Rust for speed, safety, and reliability.

Rust 2024 Edition Multi-Provider LLM WebSocket Protocol v3 Active Development

Features

A modular, trait-based architecture across 18 specialized crates.

Done

Gateway Server

WebSocket and HTTP server with JSON-RPC protocol, connection handshake, authentication, and embedded web UI with dark/light theme.

Done

Agent Runtime

Agent loop with streaming completions, tool calling (up to 25 iterations), system prompts, auth profiles, and multi-agent routing.

Done

Tool Execution

Shell command execution with timeout control, output limits, multi-layer security policies, allowlists, and approval workflows.

Done

OAuth & Auth

PKCE-based OAuth flow, device pairing with PKI, token storage with refresh, role-based access control with scoped permissions.

Done

Session Management

JSONL-based sessions with file locking, routing by peer/guild/team/account/channel, compaction, and history retrieval.

Done

Configuration

Multi-format config (TOML, YAML, JSON), environment variable substitution, provider enable/disable, get/set via gateway API.

Done

CLI

Full command suite: gateway, agent, channels, send, sessions, config, models, auth, onboard wizard, and doctor diagnostics.

Done

Cron Scheduling

Cron expression parsing, persistent job storage, isolated agent execution, and execution history tracking.

Partial

Plugin System

Comprehensive plugin API for tools, channels, hooks, providers, commands, and routes. Disk loading in progress.

Partial

Memory / RAG

Vector storage with SQLite + sqlite-vec, hybrid keyword and semantic search. Full backend completion underway.

Partial

Browser Automation

Chrome DevTools Protocol integration with profile isolation, screenshots, snapshots. CDP pool manager in progress.

Partial

Media Pipeline

Download, storage, MIME detection, image resize, HTTP serving, TTL-based cleanup. Audio transcription coming soon.

Planned

Messaging Channels

Telegram, Discord, Slack, WhatsApp, SMS, Email, Signal. Plugin architecture is ready, channel implementations next.

Planned

Web Tools

Web fetch and web search tools for agents. Enables LLMs to browse and retrieve information from the web.

Planned

Image Generation

AI image generation tool integration for agents to create images on demand.
